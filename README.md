# Eye Pupil Controlled Wheelchair

## Overview
This project implements an eye gaze controlled smart wheelchair for physically disabled individuals using Raspberry Pi, OpenCV, and Python.

---

## Features
- Eye movement detection
- Real-time webcam tracking
- Wheelchair directional control
- Motor driver integration
- Obstacle detection support

---

## Hardware Components
- Raspberry Pi 4 Model B
- L298N Motor Driver
- Webcam
- 12V DC Motors
- Battery
- Wheels and sprocket system

---

## Software Used
- Python 3.8
- OpenCV
- VS Code
- Thonny

---

## Folder Structure

```text
hardware/     -> Circuit diagrams and hardware details
software/     -> Python and OpenCV code
simulation/   -> Output screenshots and testing
documents/    -> Project report and presentation
images/       -> Project photos
```

---

## Working Principle
The webcam captures the eye movement of the user. OpenCV processes the eye position and Raspberry Pi sends movement commands to the motor driver for wheelchair navigation.
